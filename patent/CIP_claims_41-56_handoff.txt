WHAT THE PROVISIONAL COVERS (Claims 1–40)
═══════════════════════════════════════════════════════════════

The filed provisional covers the cosmological-scale UHA encoding:
- Linear radial normalization: s₁ = r / R_H (horizon-normalized)
- Fixed 21-bit Morton Z-order encoding of 3D position
- CosmoID: cryptographic hash of cosmological parameter set (H₀, Ω_m, Ω_Λ, etc.)
- TLV address format (type-length-value, self-decoding)
- Friedmann integral for exact horizon radius R_H(a)
- Single operational mode (cosmological)

Core invention: any spatial position in the universe encoded as a
self-decoding, H₀-independent address tuple. Two addresses with the
same CosmoID are directly comparable regardless of observer frame.

═══════════════════════════════════════════════════════════════
NEW MATTER IN THE CIP (Claims 41–56)
═══════════════════════════════════════════════════════════════

Six categories of new matter extend UHA from cosmological scales
down to 10 cm terrestrial precision:

CATEGORY A — Scale Dilation (Claims 41–42)
  Adaptive bit depth N(a) = ⌈log₂(a · R_H(a) / ε_target)⌉
  Preserves physical precision floor ε_target across all cosmic epochs.
  N clamped to [10, 63]. ε_target encoded in CosmoID → self-decoding.

CATEGORY B — Multi-Tier Radial Bands (Claims 43–45)
  Partitions [0, R_H] into K=4 tiers:
    Tier 0 (Local):        r < 1 Mpc      (Earth, LEO, solar system)
    Tier 1 (Stellar):      1–100 Mpc      (nearby stars, clusters)
    Tier 2 (Galactic):     100–1000 Mpc   (Milky Way, Local Group)
    Tier 3 (Cosmological): 1000 Mpc–R_H   (large-scale structure)
  Composite index: (tier_bits << 63) | morton_index_within_tier
  Per-tier bit depth: higher near-field, lower cosmological.

CATEGORY C — Logarithmic Radial Normalization (Claims 46–47)
  s₁_log = log(1 + r/r₀) / log(1 + R_H/r₀)
  Single formula spans 10 cm → Hubble horizon without regime switching.
  Default r₀ = 1 AU. Concentrates quantization at near-field distances.
  r₀ encoded in CosmoID → self-decoding.

CATEGORY D — Adaptive Bit Depth by Spatial Regime (Claims 48–49)
  N(r) = max(N_min, min(N_max, N_base - β · ⌊log₁₀(r / r_ref)⌋))
  Near-field gets N=63, cosmological gets N=21.
  N_min ≥ 10 (preserves Claim 3 minimum). All params in CosmoID.

CATEGORY E — 10 cm Precision Target (Claim 50)
  At Earth surface (r ≈ 6.371×10⁻⁹ Mpc), N=63, r₀=1 AU:
    δ_phys ≤ 0.10 m  (conservative bound; actual computed: ~1.26 cm)
  Verified formula:
    δr = r₀ · (1 + r/r₀) · (1/2^N) · ln(1 + R_H/r₀)
    = 1.496×10⁻⁷ · 1.043 · 1.084×10⁻¹⁹ · 24.12 ≈ 1.26 cm
  Comparison: GPS (WGS-84) = 3.5m without DGPS; UHA = 1.26 cm,
  cryptographically auditable, cosmology-portable.

CATEGORY F — Local-Mode / Cosmological-Mode Switch (Claims 51–52)
  Deterministic mode selection rule:
    if r < r_switch AND (H₀·r/c) < δ_threshold:
        mode = LOCAL   (R_ref = R_local, expansion term = 0)
    else:
        mode = COSMOLOGICAL  (R_ref = R_H(a), full Friedmann)
  Defaults: r_switch = 100 Mpc, δ_threshold = 0.01 (1% expansion).
  Mode + thresholds encoded in CosmoID → decoder recovers mode without
  external configuration. First system to span cm → Gpc in one address format.

ADDITIONAL CLAIMS:
  Claim 53 — Uncertainty vector σ_unc encoded inside the address tuple
              (not as external metadata). Uses N/U Algebra interval arithmetic.
  Claim 54 — Velocity vector σ_vel (proper motion / peculiar velocity)
              encoded in address. Position + velocity + cosmological context
              in one self-decoding tuple.
  Claim 55 — ξ coordinate output: ξ = d_c / R_H(a), dimensionless, H₀-independent.
              Frame-invariance property: ξ(A₁) = ξ(A₂) iff CosmoIDs identical
              or gateway-transformable. This is the coordinate used in Papers 1 & 2.
  Claim 56 — UHA + uncertainty propagation system (combines Claims 1 + 53).
              Conservative interval arithmetic on uncertainty indices.

Gaia catalog:   Velocity as external table lookup, not self-decoding

                output with CosmoID provenance and frame-invariance property

No prior system spans cm to Gpc in a single self-decoding address format
with cryptographically auditable coordinate provenance.
